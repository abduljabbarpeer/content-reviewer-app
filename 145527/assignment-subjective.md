## Scenario-Based Question

You are a SOC Analyst responsible for a newly deployed SIEM system using the ELK Stack (Elasticsearch, Logstash, Kibana, FileBeat) running on Docker.

The system is currently facing the following issues:
- Intermittent log ingestion failures due to Docker port conflicts and DNS errors  
- Logs are visible in Kibana but important suspicious activities (e.g., repeated failed logins followed by success) are not generating alerts  
- Data from multiple sources (servers, firewalls, applications) appears inconsistent and difficult to correlate  

Design a complete approach to stabilize the SIEM deployment, ensure reliable log ingestion, improve detection of security incidents, and enable effective SOC monitoring and investigation using the ELK dashboard.

Your answer should reflect a practical, step-by-step strategy covering deployment stability, log flow handling, detection improvements, and analyst usage."
